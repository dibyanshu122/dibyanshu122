# 📘 HOW TO CONVERT RESUME TO PDF & WORD FORMAT

## Step 1: Download Plain Text Resume

Go to: https://github.com/dibyanshu122/dibyanshu122/blob/main/RESUME_DIBYANSHU_SINGH_PLAINTEXT.txt

Click "Raw" button → Right-click → Save as → `Dibyanshu_Resume.txt`

---

## Step 2: Convert TXT to WORD (.docx)

### **Option A: Using Google Docs (FREE & BEST)**

1. Open Google Docs: https://docs.google.com
2. Click "File" → "Open"
3. Click "Upload" tab
4. Upload `Dibyanshu_Resume.txt`
5. Google will open it in Docs format
6. Format it:
   - Select all text: Ctrl+A (or Cmd+A on Mac)
   - Change font to: **Calibri** or **Arial**
   - Font size: **11pt**
   - Line spacing: **1.15**
   - Margins: **1 inch** on all sides

7. Make it look professional:
   - **BOLD** your name at top
   - **BOLD** section headers (PROFESSIONAL SUMMARY, EXPERIENCE, etc.)
   - **BOLD** company names and job titles

8. Export as Word:
   - Click "File" → "Download" → "Microsoft Word (.docx)"
   - Name it: `Dibyanshu_Singh_Resume.docx`

### **Option B: Using Microsoft Word Online (FREE)**

1. Go to: https://office.com
2. Sign in with your Microsoft account (or create one - FREE)
3. Click "Create" → "Blank document"
4. Paste the text from the .txt file
5. Format as described above
6. Save as `.docx`

### **Option C: Using LibreOffice (FREE & Open Source)**

1. Download LibreOffice: https://www.libreoffice.org/
2. Open the .txt file with LibreOffice Writer
3. Format the resume
4. File → Export as PDF or Save as .docx

---

## Step 3: Convert to PDF (RECOMMENDED FOR SUBMISSIONS)

### **Option A: Google Docs Method (BEST)**

1. Open your formatted resume in Google Docs
2. Click "File" → "Download" → "PDF Document (.pdf)"
3. Name it: `Dibyanshu_Singh_Resume.pdf`

### **Option B: Word to PDF**

1. Open `.docx` file in Microsoft Word or Google Docs
2. Click "File" → "Download as" or "Export" → "PDF"

### **Option C: Online Converter (if needed)**

- Go to: https://www.smallpdf.com/word-to-pdf
- Upload your `.docx` file
- Download as PDF

## Step 6: Customize Resume for Each Job Application

### **Find Keywords from Job Description**

For each job you apply to:

1. Copy the job description
2. Identify 10-15 key skills mentioned
3. Make sure these keywords appear in your resume
4. ATS scanners rank you higher when you match keywords

**Example Keywords to look for:**
- Python, FastAPI, Django
- React, Next.js, TypeScript
- PostgreSQL, MongoDB, Redis
- AWS, Docker, CI/CD
- AI, LLM, LangChain
- etc.

### **Tailor Your Professional Summary**

Change the first few lines based on the job:

**For Backend SaaS Role:**
"Backend Developer with 2+ years building scalable microservices and production AI systems. Built B9 Automation handling 100K+ daily API requests with FastAPI and PostgreSQL. Seeking Backend Engineer role to architect high-performance systems."

**For Full Stack Role:**
"Full Stack Developer specializing in SaaS platforms and AI integration. Founded B9 Automation (WhatsApp AI platform) and delivered 3 production systems. Expert in Next.js, FastAPI, and cloud infrastructure."

**For AI Developer Role:**
"AI/ML Engineer with expertise in LLMs, vector databases, and multi-agent systems. Built production AI agents with LangChain, OpenAI, and Groq. Founding developer of B9 Automation AI platform serving 100+ businesses."
